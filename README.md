# 🎓 Education Management System (Dershane Yönetim Sistemi)

![PHP](https://img.shields.io/badge/Backend-PHP-blue)
![MySQL](https://img.shields.io/badge/Database-MySQL-orange)
![Bootstrap](https://img.shields.io/badge/Frontend-Bootstrap-purple)

Dershane ve eğitim kurumları için geliştirilmiş, öğrenci takip, sınav yönetimi ve ders programı entegrasyonu sağlayan web tabanlı bir yönetim sistemidir.

## 🚀 Projenin Amacı
Bu sistem, eğitim kurumlarındaki öğrenci bilgilerinin dijitalleştirilmesi ve hem idari personelin hem de öğrencilerin sürece kolayca dahil olabilmesi için geliştirilmiştir.

## 🔑 Temel Özellikler

* **Öğrenci Portalı:** Her öğrenciye özel kullanıcı adı ve şifre ile giriş yapma.
* **Sınav Yönetimi:** Öğrencilerin sınav sonuçlarını görüntüleyebileceği dinamik panel.
* **Ders Programı:** Günlük/Haftalık ders saatlerinin görüntülendiği arayüz.
* **Yetkilendirme:** Yönetici ve Öğrenci rolleri ile veriye erişim kontrolü.
* **Veritabanı Entegrasyonu:** Öğrenci kayıtları ve notların MySQL ile güvenli yönetimi.

## 💻 Kullanılan Teknolojiler
* **Backend:** PHP
* **Veritabanı:** MySQL
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap (Duyarlı Tasarım)
* **Sunucu:** XAMPP / WAMP

## 🛠️ Nasıl Çalıştırılır?
1. Bu projeyi bilgisayarınıza indirin veya klonlayın.
2. XAMPP/WAMP sunucunuzu başlatın.
3. Proje klasörünü `htdocs` içerisine yerleştirin.
4. `database.sql` dosyasını `phpMyAdmin` üzerinden veritabanınıza import edin.
5. `config.php` dosyasında veritabanı kullanıcı adı ve şifre bilgilerinizi kendi ayarlarınıza göre güncelleyin.
6. Tarayıcınızdan `localhost/education-management-system` adresine gidin.
